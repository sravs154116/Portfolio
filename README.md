# 🌐 Sravanthi Chillakuru — Personal Portfolio

A modern, responsive developer portfolio built with **React + Vite**, featuring a stunning 3D animated background, glassmorphism UI, smooth scroll animations, and a fully functional WhatsApp-integrated contact form.

---

## 👩‍💻 About

This is the personal portfolio of **Sravanthi Chillakuru**, a final-year B.Tech Computer Science & Engineering student at Narayana Engineering College, Gudur, with a CGPA of **9.0/10**.

Passionate about **Artificial Intelligence**, **Machine Learning**, **Java Development**, and **Full-Stack Web Development**, with hands-on experience from **6 industry internships** across healthcare, cybersecurity, education, and disaster management domains.

---

## ✨ Features

- 🎨 **Glassmorphism Dark Theme** — Sleek, modern UI with deep navy and glowing blue accents
- 🌐 **Interactive 3D Background** — Animated 3D shapes using React Three Fiber & Three.js
- ⚡ **Smooth Animations** — Powered by Framer Motion with scroll-triggered transitions
- 📱 **Fully Responsive** — Optimized for all screen sizes
- 🖱️ **Custom Cursor** — Premium interactive cursor experience
- 📊 **Scroll Progress Bar** — Real-time reading progress indicator
- 💬 **WhatsApp Contact Form** — Messages sent directly to phone via WhatsApp
- 🔗 **Social Links** — GitHub, LinkedIn, Email, LeetCode, HackerRank

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or above)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/sravs154116/Portfolio.git

# Navigate to the project directory
cd Portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at `http://localhost:5173`

### Build for Production

```bash
npm run build
```
